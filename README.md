### GET a random outfit

| URL                              | Method |
| -------------------------------- | ------ |
| http://localhost:8000/surpriseme | `GET`  |

Returns a random outfit which consists of an underwear, outerwear and accessories, The data is fetched from the NEWYORKER API.
